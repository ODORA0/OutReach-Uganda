# OutReach Uganda

A modern, responsive charity website dedicated to supporting children and communities in Uganda through education, healthcare, and community development programs.

## About

OutReach Uganda is committed to transforming lives through sustainable development initiatives. Our programs focus on:

- **Education**: Providing quality education and learning resources
- **Healthcare**: Ensuring access to medical care and health services
- **Community Development**: Building sustainable communities through various initiatives
- **Child Sponsorship**: Creating lasting impact through one-on-one sponsorship programs

## Website Features

- Modern, responsive design optimized for all devices
- African-inspired aesthetic with vibrant colors and engaging visuals
- Multiple pages showcasing our mission, programs, and impact
- Interactive gallery highlighting our work
- Easy-to-use donation and contact forms
- Comprehensive documentation for donors and partners

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ODORA0/OutReach-Uganda.git
cd OutReach-Uganda
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
